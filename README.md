# artyyc-load-pieces
Script to scrape city data and load it into our firebase datastore

# To Run
```bash
 # Clone the repo
 $ git clone https://github.com/CivicTechYYC/artyyc-load-pieces \n
 
 # cd into directory
 $ cd artyyc-load-pieces
 
 # Install Dependancies
 $ npm install
 
 # Run the script 
 $ node load.js
 ```
